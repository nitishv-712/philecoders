# Product Overview

## Project Purpose
PhileCoders website — a Next.js 16 web application serving as the public-facing website for the PhileCoders organization/team.

## Key Features & Capabilities
- Server-side rendering and static generation via Next.js App Router
- Responsive UI built with Tailwind CSS v4
- TypeScript throughout for type safety
- Optimized font loading via `next/font` (Geist font family)
- Fast page transitions and automatic code splitting

## Target Users & Use Cases
- Visitors learning about PhileCoders
- Team members maintaining and extending the site
- Developers onboarding to contribute to the codebase
